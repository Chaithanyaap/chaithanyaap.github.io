Automation & Scripting

I actively build automation tools using PowerShell to reduce manual operational work and improve infrastructure reliability.

  • ESXi Patching Report Automation – Generated automated ESXi patch compliance reports and emailed them to administrators.
  • Infrastructure Inventory Report – PowerShell script to collect VM, host, CPU, memory, and storage inventory across the environment.
  • Memory Expansion Tool – Automated VM memory recommendation based on VM size and usage metrics.
  • Bulk ESXi NTP Configuration – Script to enable and configure NTP across multiple ESXi hosts where it was disabled.
  • Windows Patch Compliance Report – Script that collects Windows patch status across servers and generates compliance reports.
  • VM Snapshot Cleanup Automation – Identifies old VM snapshots and generates alerts or cleanup recommendations.
  • Disk Utilization Monitoring Script – Monitors server disk usage and triggers alerts when thresholds are exceeded.
  • Server Health Check Automation – Script that checks CPU, memory, services, and disk status for multiple servers.
